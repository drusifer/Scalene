## Key Findings (2026-07-09, `*qa uat copyable system messages`)
- Lesson for future UAT of any "here's a command to copy-paste" feature: don't just check the string's *content* is correct — actually paste it into a real shell. `shlex.quote()` applied to every argument except one (a hardcoded placeholder) is exactly the kind of asymmetry that's invisible in a code read but breaks immediately on first real use. Artifact-based verification protocol extends to "run the exact thing a user would run," not just "read the code that builds it."
